## Instalação
### Pré Requisitos
Git, node.js, yarn

### Instalar Dependências
```bash

# Clone o repositório
$ git clone https://github.com/3salles/Proffy.git

# Abra a pasta do servidor
$ cd Proffy/server

# Instale as dependências
$ yarn start

# Acesse a porta 
http://localhost:3333/
```
