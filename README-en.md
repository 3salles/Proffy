## Installation
### Pre Requieuments
Git, node.js, yarn

### Install Dependencies
```bash

# Clone repository
$ git clone 

# Open server folder
$ cd Proffy/server

# Install dependencies
$ yarn start

# Acess port link
http://localhost:3333/
```
